Detailed Description

make a class polymorphic you can:

  • Add Q_CLASSINFO("polymorphic", "true") to its definition
  • Globally force polyphormism (See QJsonSerializer::polymorphing in the doc)
  • Set a dynamic property: setProperty("__qt_json_serializer_polymorphic", true);

By default, the objectName property of QObjects is not serialized (See keepObjectName)

  1. By default, the JSON null can only be converted to QObjects. For other types the conversion fails (See allowDefaultNull)
